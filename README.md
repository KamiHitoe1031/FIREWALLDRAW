# ファイアウォール・ドロー

お絵かき×タワーディフェンス ゲーム

## 起動方法

### 方法1: ローカルサーバーを使用（推奨）

JSONファイルを正しく読み込むにはローカルサーバーが必要です。

**npx http-server を使用:**
```bash
cd firewall-draw
npx http-server -p 8080 -c-1
```
ブラウザで http://127.0.0.1:8080 を開く

**npx live-server を使用（ホットリロード付き）:**
```bash
cd firewall-draw
npx live-server --port=8080
```
自動的にブラウザが開きます

**VS Code Live Server 拡張機能:**
1. VS Codeで「Live Server」拡張機能をインストール
2. index.htmlを右クリック → 「Open with Live Server」

### 方法2: 直接開く（file://プロトコル）

`index.html` をダブルクリックまたはブラウザにドラッグ＆ドロップ。

※この方法ではJSONファイルが読み込めませんが、内蔵のデフォルトデータで動作します。

## 操作方法

- **マウスドラッグ**: 壁を描く
- **ポーズボタン（||）**: 一時停止・音量調整

## ゲームルール

1. 画面中央のCPUをウイルスから守る
2. マウスで光の壁を描いてウイルスを阻止
3. 壁に触れたウイルスはダメージを受ける
4. ウイルスがCPUに到達するとHPが減る
5. 全ウェーブを乗り越えればステージクリア

## キャラクター選択

ステージ開始前のブリーフィング画面で5種類のCPUキャラクターを選択できます。
各キャラクターには壁の数・持続時間・CPU HP・壁ダメージに個性があります。

| キャラ | タイプ | 特徴 | コスト |
|--------|--------|------|--------|
| スタンダード | バランス型 | 標準値 | 無料 |
| フォートレス | 耐久型 | HP+3、壁持続-1秒 | 200 |
| マルチコア | 壁攻勢型 | 壁+1、HP-2 | 300 |
| クロノ | 持久型 | 壁持続+2.5秒、壁-1 | 400 |
| ストライカー | 攻撃型 | ダメx1.4、HP-2 | 500 |

※スタンダード以外はコインで解放

## 壁の仕様

- 同時に存在できる壁: 3本（4本目を描くと最古が消える）
- 壁の持続時間: 5秒
- 最小長さ: 50px（短すぎると無効）
- 最大長さ: 300px（超えると自動で切られる）
- クールダウン: 0.5秒

## 音量調整

ゲーム中にポーズ（||）を押すと、5段階の音量調整ボタンが表示されます。

## 画像の追加

画像ファイルを指定フォルダに配置してリロードすると自動で反映されます。

| ファイル | 配置場所 |
|---------|---------|
| 敵スプライト | assets/images/enemies/ |
| CPUキャラクター | assets/images/cpu/ |
| UI画像 | assets/images/ui/ |
| 背景画像 | assets/images/backgrounds/ |
